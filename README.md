# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases. The objective is to identify spending patterns, customer segments, product preferences, and subscription trends to support business decision-making.

## Dataset
- 3,900 rows and 18 columns
- Includes customer demographics, purchase details, discounts, ratings, and shopping behavior
- Missing values in `review_rating` handled using median imputation

## Technologies Used
- Python (Pandas, NumPy)
- PostgreSQL
- Power BI

## Data Preprocessing
- Cleaned and standardized column names
- Handled missing values
- Created new features such as `age_group`
- Removed redundant columns
- Loaded cleaned data into PostgreSQL

## SQL Analysis
Performed business analysis including:
- Revenue by gender
- Subscriber vs non-subscriber spending
- Top-rated products
- Customer segmentation
- Revenue by age group
- Discount impact analysis
- Top products per category

## Dashboard
Created an interactive Power BI dashboard to visualize customer behavior, revenue trends, and product performance.

## Business Recommendations
- Improve subscription benefits
- Introduce loyalty programs
- Optimize discount strategies
- Focus marketing on high-value customer groups
